# weflow

## Packages

|Name|description|
|-|-|
|@weflow/cli|cli 工具主要模块，处理用户的 cli 交互|

|@weflow/webpack-plugin|通过 webpack 构建小程序的 webpack 插件|
|@weflow/wxml-loader|webpack 的 wxml loader|
|@weflow/wxss-loader|webpack 的 wxss loader|
